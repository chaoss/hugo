### **Reminder! Schedule is Published for CHAOSScon NA 2023** {.has-vivid-red-color.has-text-color.wp-block-heading}

The CHAOSScon planning committee has released the schedule for CHAOSScon NA 2023, to be held&nbsp;in the **afternoon of May 9, 2023** in conjunction with Open Source Summit North America in Vancouver BC, Canada. You can view the schedule [here][1].

In-person registration is $20 USD and registration for OSSNA is a pre-requisite. Live streaming will be available for free, but registration for OSSNA virtual is required. You can register for both events on the [OSSNA registration page][2]. [Sponsorship][3] is also available, so if you&#8217;re interested in sponsoring, feel free to reach out.

Hope you can join us, either in-person or virtually, and don&#8217;t forget to register! 



### **Change to Communications Working Group Meeting Cadence** {.has-vivid-red-color.has-text-color.wp-block-heading}

The Communications Working Group will now be meeting **_every other Thursday at 9:00 am US Central/Chicago time for 50 minutes._** There will no longer be meetings on Mondays. If you would like to join us you can subscribe to the [CHAOSS calendar][4] for login details or you can join us anytime asynchronously in the [#wg-communications Slack channel][5] 

### **New Augur Release!**  {.has-vivid-red-color.has-text-color.wp-block-heading}

The Augur team is happy to announce that there is a new release available. From Sean Goggins: 

<blockquote class="wp-block-quote is-layout-flow wp-block-quote-is-layout-flow">
  <p>
    This release makes a hosted version of Augur fully enabled. The next release will add to these capabilities by implementing a “fairness algorithm” so that data collection for new accounts is balanced and somebody asking for 1,000 repositories does not delay collection for somebody requesting 10. Further, the next release will consider the size of the new repositories requested when balancing individual user collection priorities. Integration with <a href="https://github.com/oss-aspen/8Knot" target="_blank" rel="noreferrer noopener">https://github.com/oss-aspen/8Knot</a> is also improved, enabling collections of related repositories to be shared by all of the users of an 8Knot instance. 
  </p>
</blockquote>

If you&#8217;d like to download the latest release you can find it [here.][6]

### **New Blog Post &#8211; Starter Project Health Metrics Model** {.has-vivid-red-color.has-text-color.wp-block-heading}

Getting started with tracking and measuring open source community health can be tricky! CHAOSS released a metrics model that can help. If this describes the predicament you&#8217;re currently in, we recommend checking out our [blog post][7] and [Starter Project Health Metrics Model][8] for guidance.
