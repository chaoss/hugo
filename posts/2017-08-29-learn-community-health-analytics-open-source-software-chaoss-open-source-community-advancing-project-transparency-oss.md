 <span style="font-weight: 400;">Learn more about </span>[**Community Health Analytics Open Source Software (CHAOSS):  An Open Source Community for Advancing Project Transparency OSS**][1] **on Monday Sept. 11 at Open Source Summit North America.  **<span style="font-weight: 400;">CHAOSS is a new Linux Foundation project aimed at </span><span style="font-weight: 400;">producing integrated, open source software for analyzing software development, and definition of standards and models used in that software in specific use cases and establishing implementation-agnostic metrics for measuring community activity, contributions, and health. </span><span style="font-weight: 400;">The CHAOSS community will help improve transparency of key project metrics, contributing to improve the project itself as well as helping third parties make informed decisions when engaging with projects.</span>
